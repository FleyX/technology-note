基于`create-react-app`进行开发，选择这个框架有以下两个原因：

- 从头开始配置 webpack 太复杂，不用了解的这么细致。
- 也不使用`antd pro`,`umi.js`这类开箱即用的，隐藏了太多细节，也不考虑。
